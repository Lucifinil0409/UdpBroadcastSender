# UdpBroadcastSender
15连屏的主机控制程序


主机用于在局域网中广播控制指令。
广播“OPEN”，“CLOSE”，“COPY”，“SHUTDOWN”指令。
（本来只是个测试用的程序，没想到后来就直接用了）
